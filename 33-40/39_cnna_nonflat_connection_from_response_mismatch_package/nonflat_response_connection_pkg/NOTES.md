# Notes

## Reading rule

The exact metric transport

```text
T_ij = G_j^{-1/2} G_i^{1/2}
```

is included only as a gauge audit.  It can telescope by construction, so zero holonomy there is not a surprise and must not be sold as a positive result.

The intended nonflat-connection diagnostic is the Wilson-loop family built from same-label response-mismatch generators:

- `metricWilson`: generated by metric differences `G_j - G_i`;
- `KWilson`: generated by plaquette-skew differences `K_j - K_i`;
- `GcommWilson`: generated by response-metric commutators `[G_i,G_j]`;
- `mixedWilson`: generated by a mixed metric/skew response expression.

A positive Stufe-4 gate would require real-growth/full/live to exceed identical-history, diagonal/trace and random controls in a way that is not reproduced by symmetrized birth.

## Observed status in the included L5/L6 runs

The local parent-fan J sector remains present (`K != 0`).  Symmetrized birth suppresses `K` strongly and diagonal/trace reductions kill it.

However, the same-label response-mismatch Wilson loops are extremely small.  The exact metric transport is flat to numerical precision, and the Wilson diagnostics do not beat the random same-level control.  Therefore this package is a negative Stufe-4 test.
