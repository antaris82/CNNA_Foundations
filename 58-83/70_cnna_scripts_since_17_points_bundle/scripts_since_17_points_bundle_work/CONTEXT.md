# Kontext

Dieses Paket ist eine Skript-Sammlung seit dem 17-Punkte-Pivot der CNNA-Provenienzlinie.
Der Pivot verschob die Suche von einer direkten lokalen `J/i`-Suche zu einer zweiachsigen Wachstums-/Randwertstruktur:

- longitudinale root/front-, record/live-, retarded/advanced-Polung,
- transversale Wachstums-/Regionenbildung,
- deterministischer Leitwertversatz der Geschwister,
- echte Schur/DtN-Randantworten,
- irreversible Birth+Relaxation-Mischung,
- coarse-grained Operator-Vorstrukturen.

Die Sammlung enthält Skripte und Markdown-Dokumente aus den erzeugten Paketen; große CSV/JSON-Ausgabedaten sind absichtlich nicht Teil dieses Sammelpakets.
